## ğŸ’» Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to have `Ruby` installed on your computer

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:tobuya/morse-decoder.git
```

### Install [optional]

Install inters for this project with:

```sh
  cd morse-decoder
  bundle install
```

### Usage

To run the project, execute the following command:

```sh
  irb -r ./morse_decoder.rb
```

> To decode a message:

```sh
  decode(<youre_morse_code>)
```

Example:

```sh
  decode(".-   -... --- -..-   ..-. ..- .-.. .-..   --- ..-.   .-. ..- -... .. . ...")
   => "A BOX FULL OF RUBIES"
```
